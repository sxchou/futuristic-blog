## 🎯 项目简介

**Futuristic Blog** 是一款面向开发者的现代化个人博客系统，采用前后端分离架构，前端使用 Vue 3 + TypeScript + Vite，后端使用 Python + FastAPI。系统设计注重用户体验、性能优化和代码质量，适合作为个人技术博客、知识分享平台使用。

### ✨ 核心特性

- 🔐 **完整的用户认证系统** - 支持邮箱注册、OAuth 第三方登录（Google/GitHub/X/微信/QQ）
- 📝 **强大的文章管理** - Markdown 编辑、代码高亮、文件附件、自动别名生成
- 💬 **互动评论系统** - 支持 Markdown、嵌套回复、邮件通知、审核机制
- 🎨 **现代化 UI 设计** - 响应式布局、暗黑模式、玻璃态效果、流畅动画
- 📊 **数据可视化** - 访问统计、趋势图表、排行榜、实时数据展示
- 🛡️ **角色权限管理（RBAC）** - 细粒度权限控制、多角色支持、权限继承、操作审计
- 📜 **系统日志导出** - 操作日志/登录日志/访问日志导出 Excel、进度追踪、取消导出
- 🔧 **灵活的后台管理** - 内容管理、用户管理、系统配置、存储管理
- 📦 **文件存储管理** - Supabase 存储、文件预览、批量管理、存储统计
- 📢 **公告系统** - 多类型公告、优先级排序、启用/禁用控制

## 🏗️ 技术架构

### 前端技术栈

| 技术                    | 说明                                   |
| --------------------- | ------------------------------------ |
| Vue 3                 | 渐进式 JavaScript 框架，采用 Composition API |
| TypeScript            | 类型安全，提升开发体验                          |
| Vite                  | 下一代前端构建工具，极速热更新                      |
| Pinia                 | Vue 官方状态管理库                          |
| Vue Router            | 单页面应用路由管理                            |
| Tailwind CSS          | 原子化 CSS 框架                           |
| Marked + Highlight.js | Markdown 解析与代码高亮                     |
| ECharts               | 数据可视化图表库                             |

### 后端技术栈

| 技术                | 说明           |
| ----------------- | ------------ |
| Python 3.11+      | 现代化编程语言      |
| FastAPI           | 高性能异步 Web 框架 |
| SQLAlchemy        | ORM 数据库工具    |
| PostgreSQL (Neon) | 云原生数据库       |
| Supabase Storage  | 对象存储服务       |
| JWT               | 用户认证令牌       |
| Resend / SMTP     | 邮件发送服务       |

### 部署架构

| 层级      | 服务               | 说明                    |
| ------- | ---------------- | --------------------- |
| 🎨 前端   | Vercel / Netlify | 静态托管，全球 CDN 加速        |
| ⚙️ 后端   | Render / Railway | API 服务，自动扩容           |
| 🗄️ 数据库 | Neon             | PostgreSQL，Serverless |
| 📦 存储   | Supabase Storage | 文件存储，CDN 加速           |

**数据流向**：前端 → 后端 API → 数据库 / 文件存储

## 🚀 快速部署

### 环境要求

- Node.js 18+
- Python 3.11+
- PostgreSQL 数据库
- Supabase 存储桶（可选）

### 本地开发部署

```bash
# 克隆项目
git clone https://github.com/sxchou/futuristic-blog.git
cd futuristic-blog

# 后端
cd backend && cp .env.example .env
pip install -r requirements.txt
uvicorn app.main:app --reload

# 前端
cd ../frontend && cp .env.example .env
npm install && npm run dev
```

### 生产环境部署

```bash
# 前端构建
cd frontend && npm run build

# 后端启动
cd backend && uvicorn app.main:app --host 0.0.0.0 --port 8000
```
