# Screenshots Directory

This directory contains project screenshots for the README.

## Required Screenshots

Please add the following screenshots:

1. **home.png** - Homepage screenshot
2. **article.png** - Article detail page screenshot
3. **admin.png** - Admin dashboard screenshot
4. **editor.png** - Article editor screenshot
5. **dark.png** - Dark mode screenshot
6. **mobile.png** - Mobile responsive screenshot

## Recommended Tools

- **Windows**: [ShareX](https://getsharex.com/), [Snipaste](https://www.snipaste.com/)
- **macOS**: CleanShot X, Kap (for GIF)
- **Cross-platform**: [ScreenToGif](https://www.screentogif.com/) (for animated GIF)

## Tips

- Use 1920x1080 or 1280x720 resolution
- Keep file size under 1MB
- Use PNG format for static images
- Use GIF format for animations
- Show the best features of your project
